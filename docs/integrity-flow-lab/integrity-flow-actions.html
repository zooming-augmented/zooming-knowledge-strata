Integrity Flow Actions

小さな一歩から、人生を自分に取り戻す — アクション候補一覧

自分に合う一歩を1つ選んで、5〜15分で試してみてください。
完璧にやる必要はありません。公開しなくても大丈夫です。
起点は Direction（仮の最終形）かときめき。どちらもなければ A01 から。

起点の見つけ方

Direction（仮の最終形）：「こういう方向に進みたい」という仮の目的地。正確でなくていい。曖昧でOK。

ときめき：内側から出てくる「これ、やってみたい」という感覚。他人の期待や義務感ではなく、自分の反応を起点にする。

どちらもなければ A01 から始めてください。AIと話しながら見つかります。

レベル設計

レベル対象AI活用
初級（共通）誰でも。AI初心者OK相談・補助・指示
中級（職種別）エンジニア・バックオフィス・製造などコンテキスト設計
上級（個人最適化）人事・マーケター・SWエンジニアなど自動化・パイプライン

上から順にやる必要はありません。一部だけ選んで構いません。

初級フロー

Direction またはときめき → リサーチする → アクションを選ぶ → 動けない理由（ボトルネック）を特定する → 5〜15分の計画を立てる → 実践する → 観察・振り返り → 次の一歩を決める → 公開する（任意） 成功の定義：次の一歩が決まり、次の循環に入れたら成功

まず始める

A01. AIに「いま気になっていること」を話す

目的：頭の中のもやもやを外に出す

やること：ChatGPTやClaudeに、音声または文章で3分だけ話す

例：「最近気になっていることを整理したい。結論を急がず、問いを一緒に見つけて」

記録：気づいた言葉を1つだけ残す

A02. 今日の小さな一歩をAIと決める

目的：情報収集で止まらず、行動に移る

やること：Direction・ときめき・今の状態・使える時間を伝えて、5〜15分の行動候補を出してもらう

例：「今日は疲れているけど、5分だけ進めたい。私に合う一歩を3つ出して」

記録：選んだ一歩と、選んだ理由

A03. 「何を知るか」より「何のために動くか」を確認する

目的：AI情報の洪水で迷子にならない

やること：新しいAIニュースやツールを見た時に、自分の目的と関係するかを確認する

例：「この情報は、私の生活・仕事・家族・発信のどれに効く？」

記録：今はやらないことも書く

A04. 動けない理由（ボトルネック）を1つ見つける

目的：動けない理由を特定して、小さく崩す

やること：やりたいことに対して、時間・不安・情報不足・技術不足・公開抵抗のどれが詰まりかをAIと確認する

例：「この行動が進まない理由を、ボトルネックとして3つに分解して。最初に崩すならどれ？」

記録：今日崩すボトルネック / 崩し方

Life Flow — 生活を整える

L01. 睡眠・食事・運動のどれか1つを観察する

目的：AI実践を続ける土台を整える

やること：今日の生活の中で、乱れているものを1つだけ見る

例：「最近の睡眠を整えるために、今日できる一歩を一緒に考えて」

記録：今日できたこと / 明日変えること

L02. 家族や身近な人との時間を1つ守る

目的：AI活用が生活を削る方向に進まないようにする

やること：15分だけ、スマホを置いて会話・家事・散歩などをする

記録：やってよかったこと

L03. 自分のときめきセンサーを観察する

目的：dutyに押されて鈍った感覚を取り戻す

やること：今日少しワクワクしたことを1つ書く

例：「今日のこの反応は、本物のときめきか、短期的な刺激かを一緒に見て」

記録：ワクワクした理由の仮説

Focus Flow — 集中して作る

F01. 1つのテーマを15分だけ深掘りする

目的：情報収集を、自分の問いに結びつける

やること：気になるテーマを1つ選び、AIと壁打ちする

例：「このテーマについて、私が発信するならどこに独自性がある？」

記録：使えそうな問い・タイトル・一文

F02. AIに調べてもらう前に、自分の仮説を書く

目的：AI任せの整理にしない

やること：調査前に、自分はどう思っているかを3行書く

例：「調査前の仮説を読んで、確認すべき論点を出して」

記録：調査前後で変わったこと

F03. 下書きを完成させずに公開候補へ進める

目的：完璧主義で止まらない

やること：70点の下書きを、公開候補として保存する

例：「この下書きを、完成版ではなく進行中の記録として読める形に整えて」

記録：まだ不安な点 / 公開してもよい点

Integrity Flow — 外へ流す

I01. 今日の実践を3行で残す

目的：行動を一次価値にする

フォーマット：

  • 今日やったこと：
  • 気づき：
  • 次にやること：

例：「この3行を、noteやXに出せる短い観察文に整えて」

I02. 公開してよい部分だけ外に出す

目的：公開のハードルを下げる

やること：privateな思索から、公開してもよい一文だけ抜き出す

例：「このメモから、公開してよい範囲の短文を3案作って」

記録：出した一文 / 出さなかった理由

I03. 100円でプロセスを売る実験をする

目的：行動を続ける仕組みを作る

やること：noteで、完成品ではなく「試行錯誤のストーリー」を100円で出す

例：「この記事を、役立つ情報の販売ではなく、プロセス共有として誠実に説明して」

記録：出した理由 / 次にやること

tune our flow — 小さな一歩を、集めよう
次の一歩が決まり、次の循環に入れたら成功です。
