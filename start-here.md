# Start Here — このプロジェクトの背景と目指すところ

---

## 私は何者か

会社員。30代後半。妻と娘とパグ2匹と暮らしています。
仕事は機能安全の領域。趣味はサックスと家庭菜園。

「自分の人生をちゃんと設計したい」という気持ちは昔からあったけれど、
日々の忙しさに流されてきた部分が正直あります。

2023年ごろから生成AIを使い始め、2025年にClaude Codeに出会い、
**自分の人生設計のパートナーとしてAIを使う**ことを本格的に始めました。

---

## 何をやっているのか

### Life Flow の設計

生活リズムの安定なしに、深い仕事も家族との時間も成立しない。
だから一番最初に手をつけたのが「Life Flow」です。

具体的には：
- 毎日の記録（睡眠・体重・家族との時間・メンタルなど）
- 7:00業務シャットダウン・19:00終業という自己ルール
- Google Sheetsでのダッシュボード化
- AIとの毎日の会話による振り返りと改善

これをAIとともに設計・記録・改善しています。

### AI × 生活設計 = プロセスエコノミー

「結果を発信する」のではなく「プロセスを発信する」。

うまくいったことだけでなく、失敗したこと・やり直したこと・迷ったこともすべて記録します。
なぜなら、**リアルなプロセスにこそ再現性のある価値がある**と思っているからです。

## 目指しているところ

### 個人レベル
自分の人生を自分で設計できる状態をつくる。
仕事・家族・健康・趣味・社会貢献のすべてが、少しずつ前に進んでいる感覚。

### コミュニティレベル
同じように「設計したいけどどこから始めればいいかわからない」人の参考になること。
将来的には、環境の立ち上げ方も含めて共有し、一緒にやれる場をつくりたい。

### 社会レベル
**maximize Σ Integrity** — 個人の誠実な実践が積み重なって、社会のIntegrityが上がっていく。
これが私の長期的な目標です。

## この場所の使い方（現在の想定）

| 場所 | 役割 |
|------|------|
| **このリポジトリ（GitHub）** | 実践の記録・地層を積む・技術的な仕組みの公開 |
| **note（予定）** | ストーリーとして整理・無料＋有料で届ける |
| **Zenn（予定）** | 技術的な設計・コード・構成の解説 |
| **X / YouTube（将来）** | 発見・広報・マーケティング |

今はGitHubでの記録が中心です。
コンテンツが積み上がったら、note・Zennへの展開を始めます。

## 一緒にやってみませんか？

まだ「参加の仕組み」は整っていません。
でも、このプロジェクトを見て「自分もやってみたい」と思ったなら：

- GitHub の Watch / Star をしておくと、更新が届きます
- 将来的に、環境構築ガイドと参加の場（Discord予定）を用意します

**正直に言うと、私もまだ走り始めたばかりです。**
完成形を見せるのではなく、走りながら見せていくつもりです。
