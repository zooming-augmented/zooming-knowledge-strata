# Knowledge Strata

ここは、Integrity Flow Lab. で育てている探究のうち、他者が参照・再利用できる形に整えた公開層です。

完成した答えを並べる場所ではありません。実践、観察、振り返りから残ったものを、少しずつ辿れる形にしていきます。

privateな記録、生の対話、作業中の下書きはここには置きません。この層にあるのは、公開しても誰かの次の一歩に役立つと判断したものだけです。

## この地層の4つの棚

### Framework

探究を支える用語、原則、関係図です。世界観そのものを語るのではなく、読み手が参照できる概念として整えます。

### Experiments

日々の生活や仕事に移せる、小さな実践と観察です。うまくいったことだけでなく、試行の条件や学びも残します。

### Research

判断や実践の土台になる公開調査・レポート・参考資料への入口です。

### Stories

noteで育つ連載から、公開安全な形で抽出した要約と関連導線です。note本文を複製する場所ではありません。

## 読み方

- 人生後半をAIとともに設計する探究の背景は、[Integrity Flow Lab.](../docs/index.html) から辿れます。
- 物語として読みたい場合は、[note記事マップ](../docs/notes/index.html) へ進んでください。
- 調査・実験・概念を参照したい場合は、この `strata/` 以下を棚ごとに辿ってください。各棚は、公開できる内容が育った時に追加されます。

## 使い方と公開基準

このリポジトリは、GitHub上で読むだけでなくcloneして、履歴や差分とともに参照できます。

```bash
git clone https://github.com/zooming-augmented/zooming-knowledge-strata.git
```

各Markdownには、必要に応じて `title`、`kind`、`status`、`updated`、`related` を付けます。ここに置くのは `published` のものだけです。

公開前には、長期性・双方向価値・再現性・誠実さを確かめます。迷う内容は開かず、private側に残します。
